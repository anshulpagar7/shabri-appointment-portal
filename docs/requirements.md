# Shabri Smart Appointment System - Requirements

## Stakeholder

Adivasi Vikas Bhavan (Government of Maharashtra)

## User Roles

### Citizen

* Scan QR Code
* Book Walk-in Appointment
* Book Future Appointment
* Select Officer
* Select Purpose
* Select Date and Time
* Receive Appointment ID
* Track Appointment Status

### Staff

* View Today's Appointments
* Add Walk-in Visitors
* Mark Visitor Status
* Cancel Appointment
* Reschedule Appointment
* Manage Holidays
* Block Slots

### Madam / Officer

* View Daily Schedule
* View Current Visitor
* View Next Visitor
* Mark Meeting Complete
* Mark Delay

## Notifications

* Booking Confirmation
* Cancellation Notification
* Reschedule Notification
* Delay Notification

## Integrations

* WhatsApp
* SMS
* Google Calendar

## Languages

* English
* Marathi
* Hindi
